üìö Glosario de Funciones y Estructuras

Diccionario {}

diccionario = {}

Estructura de datos que almacena pares clave-valor. En este caso, las claves son las palabras y los valores son las frecuencias (n√∫mero de veces que aparece cada palabra). Los diccionarios son ideales para conteos porque permiten acceso r√°pido y actualizaci√≥n eficiente.

Bucle for

for elemento in lista:

Estructura de control que itera sobre cada elemento de una secuencia (en este caso, cada palabra en la lista de tokens). Permite procesar cada palabra individualmente para contar su frecuencia.

Condicional if-else

if condicion: ... else: ...

Estructura que permite tomar decisiones en el c√≥digo. Si la palabra ya existe en el diccionario, incrementa su contador en 1. Si no existe, la agrega al diccionario con un valor inicial de 1. Esta l√≥gica es fundamental para el conteo de frecuencias.

Operador in

elemento in diccionario

Verifica si una clave existe en el diccionario. Devuelve True si la palabra ya fue agregada anteriormente, o False si es la primera vez que aparece. Es esencial para evitar errores al intentar incrementar valores que no existen.

Incremento +=

variable += 1

Operador de incremento que suma 1 al valor actual de una variable. Es equivalente a escribir "variable = variable + 1". Se utiliza para aumentar el contador cada vez que se encuentra una palabra repetida.
