# Chatbot con an√°lisis de sentimientos usando TextBlob y Deep-Translator
import random
from textblob import TextBlob
from deep_translator import GoogleTranslator

# Inicializar traductor
traductor = GoogleTranslator(source='es', target='en')

# LISTAS DE SALUDOS Y RESPUESTAS
saludos = ["hola", "que tal", "buenos dias", "buenas tardes", "buen dia"]
respuesta_saludos = ["hola como estas?", "como puedo ayudarte?", "hola como te va", "que hay?"]
despedida = ["adios", "bye", "gracias", "nos vemos", "hasta luego", "salir"]
respuesta_despedida = ["adios", "hasta pronto", "fue un placer ayudarte", "cuidate", "nos vemos luego", "bye"]

# LISTAS PARA AN√ÅLISIS DE SENTIMIENTOS
pregunta_sentimiento = ["como estas", "como te sientes", "que tal estas", "como andas"]
respuesta_feliz = ["Me alegra que estes feliz!", "Que bueno que te sientas bien!", "Genial! Sigue asi!",
                   "Excelente que estes contento!"]
respuesta_triste = ["Lamento que estes triste", "Animo, todo mejora!", "Entiendo como te sientes",
                    "Espero que te sientas mejor pronto"]
respuesta_neutral = ["Entiendo", "Ok, gracias por compartir", "Comprendo como te sientes"]

# Mensaje de bienvenida del chatbot
print("Hola soy Chato, tu Chatbot de confianza, si deseas terminar escribe 'salir'")

while True:
    usuario = input("Tu: ").lower().strip()

    if usuario in saludos:
        print("Chatbot:", random.choice(respuesta_saludos))

    elif usuario in pregunta_sentimiento:
        print("Chatbot: Cuentame, ¬øcomo te sientes? (feliz o triste)")

    elif usuario in despedida:
        print("Chatbot:", random.choice(respuesta_despedida))
        break

    else:
        # Analizar sentimiento con TextBlob y Deep-Translator
        try:
            # Traducir al ingl√©s
            texto_ingles = traductor.translate(usuario)

            # Analizar sentimiento
            analisis = TextBlob(texto_ingles)
            polaridad = analisis.sentiment.polarity

            print(f"Chatbot: [Analizando sentimiento... Polaridad: {polaridad:.2f}]")

            # Responder seg√∫n el sentimiento
            if polaridad > 0.2:
                print("Chatbot:", random.choice(respuesta_feliz))
            elif polaridad < -0.2:
                print("Chatbot:", random.choice(respuesta_triste))
            else:
                print("Chatbot:", random.choice(respuesta_neutral))

        except Exception as e:
            print("Chatbot: No entendi muy bien, ¬øpuedes decirlo de otra forma?")

üìö Glosario de Funciones y Conceptos

random.choice()

random.choice(lista)

Funci√≥n del m√≥dulo random que selecciona un elemento aleatorio de una lista. Permite que el chatbot tenga respuestas variadas en lugar de repetir siempre lo mismo, haci√©ndolo m√°s natural y conversacional.

Bucle infinito que mantiene el chatbot activo esperando continuamente la entrada del usuario. Solo se detiene cuando encuentra una instrucci√≥n break (en este caso, cuando el usuario se despide).

in (operador de pertenencia)

elemento in lista

Operador que verifica si un elemento est√° presente en una secuencia. Se usa para detectar si el mensaje del usuario coincide con alguna palabra en las listas de saludos, despedidas o preguntas de sentimiento.

try-except

Manejo de excepciones que previene que el programa se detenga por errores. Si la traducci√≥n o an√°lisis falla (por problemas de conexi√≥n, por ejemplo), el chatbot responde con un mensaje de error amigable en lugar de crashear.

Listas de patrones

saludos = ["hola", "que tal", ...]

T√©cnica de pattern matching b√°sica donde definimos listas de palabras clave que el chatbot reconoce. Es simple pero efectiva para detectar intenciones comunes sin necesidad de modelos complejos de NLP.

Umbrales de polaridad

Valores de corte para clasificar sentimientos. Polaridad mayor a 0.2 indica felicidad, menor a -0.2 indica tristeza, y entre -0.2 y 0.2 se considera neutral. Estos umbrales pueden ajustarse seg√∫n la sensibilidad deseada.

Encadenamiento de m√©todos para normalizar la entrada: input() captura el texto, lower() lo convierte a min√∫sculas para comparaci√≥n consistente, y strip() elimina espacios innecesarios al inicio y final.

Estructura que prioriza las respuestas del chatbot: primero verifica saludos, luego preguntas de sentimiento, despu√©s despedidas, y finalmente hace an√°lisis de sentimientos para cualquier otro mensaje. El orden importa.
